###########################
Contents Description

latexsource.zip : the latex sources for the main paper.

latexsource_supp.zip : the latex sources for the supplement.

ordinaT_supp.pdf : the supplement pdf for the main paper.

functions.R : .R file containing r-code functions used for simulations and data applications

demo.R: .R file containing the code of a simple demo for the usage of functions.R

###########################
Note that functions.R is based on the observation model $P(y\leq \ell) = f(b_\ell+\theta)$ 
while the model in the paper is $P(y\leq \ell) = f(b_\ell-\theta)$.
Every result and argument in the paper can be applied to functions.R. 
The only difference is the sign of the theta.
